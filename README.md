## General info
Emotebox is an emoji manager for Discord. Easily manage, add, or create emojis for your servers

## Technologies
* Reactjs
* Express
* AWS S3
* Mongodb Atlas
* Node

## Setup
To run this project, install it locally
```
git clone https://github.com/khai93/emotebox.git
cd emotebox
yarn
cd client
yarn
cd ..
yarn dev
```
